# A-Robot-Delivery-path-Finding-System-implemented-using-ML-algorithms
A Robot Delivery path Finding System where the user creates an environment through web containing packages, shelfs and more and the robot would deliver them using RL.
- Frontend: HTML + JS grid interface with images for shelves, robot, etc.
- Backend: Python (Flask) server handling Q-learning logic
- Grid cells represent walls, shelves, packages, floor
- Training: Q-learning for path optimization from robot to shelf
- Visualization: Step-by-step robot movement animation on grid
